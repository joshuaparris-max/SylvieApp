## Upgrade Plan Summary for Each App

### EliasApp
- Focus: age-2 appropriate co-play, cause-and-effect interaction, calm feedback, and low-arousal engagement.
- Short-term:
  - Add explicit co-play launch prompt.
  - Build parent-visible session timer with soft/hard stop cues.
  - Improve audio labels and immediate tap feedback.
  - Add a physical play transition prompt after sessions.
- Medium-term:
  - Add a parent safety/co-play guidance panel.
  - Offer a quiet mode with reduced motion and sound.
  - Add a simple parent progress summary.
  - Add vocabulary practice cards for trucks, tools, colors, and sounds.
- Long-term:
  - Add imitation/co-play modelling prompts.
  - Add side-by-side co-play support.
  - Add a parent review mode for activity usage.
- Checklist: explicit tasks are listed, including avoiding autoplay/random feeds, removing badge-style reward language, and keeping sessions under 10 minutes.

### SylvieApp
- Focus: calm, EYLF-aligned play with movement breaks, parent invitations, and lower reward pressure.
- Short-term:
  - Add opening and closing session rituals.
  - Add movement break prompts in longer activities.
  - Embed parent peek-in cues.
  - Replace gamified reward language with narrative/informational praise.
  - Keep screens simple with large tap targets.
- Medium-term:
  - Add a guided breathing / calm-down minigame.
  - Introduce parent-mode hint prompts.
  - Add progress reflections for child and caregiver.
  - Ensure consistent calm audio labeling.
- Long-term:
  - Add “show-and-tell” features for sharing activity outcomes with a caregiver.
  - Add gentle curriculum-linked learning paths.
  - Add optional identity/wellbeing personalization.
- Checklist: explicit tasks are listed for rituals, movement breaks, co-play cues, UI simplification, and EYLF alignment.

### SylviePhonetics

- Short-term:
  - Review and sequence activities from rhyme and syllable awareness through letter-sound matching.
  - Strengthen audio-first interactions before revealing letters.
  - Add scaffolded feedback loops.
  - Add parent co-play guidance cues.
- Medium-term:
  - Add a calm breathing transition before phonics work.
  - Add child/parent progress reflection screens.
  - Add optional “sound story” narrative prompts.
  - Map activities explicitly to EYLF skills.
- Long-term:
  - Build a full phonological progression path.
  - Add a parent-facing EYLF/NSW literacy guide.
  - Add more multisensory phonics practice bundles.
- Checklist: explicit tasks for sequence, audio modelling, feedback, co-play prompts, and timing.

### Sleepy

- Short-term:
  - Add a warm/amber bedtime display mode.
  - Add low-brightness default settings.
  - Add structured routine opening and closing prompts.
  - Add a 1–2 minute breathing/calm-down prompt.
  - Add explicit parent timing and supervision notes.
- Medium-term:
  - Add a lights-out audio mode.
  - Add a parent-only story recommendation/review page.
  - Add a routine consistency guide or simple log.
  - Curate story list for low-arousal bedtime content.
- Long-term:
  - Add a nightly routine checklist.
  - Add a guided audio transition track with breath work.
  - Add a dedicated quiet night mode with minimal animation.
- Checklist: explicit tasks for visual tone, bedtime flow, audio-first options, and sleep hygiene guidance.
